# Student Performance Predictor
A Machine Learning web application that predicts student performance using **Linear Regression**. Built with Python and Streamlit, this interactive app estimates final marks based on study habits and academic history.

## Live Demo
Link: https://student-performance-predictor-jykndxefxpptjwurpvdflj.streamlit.app

## GitHub Repository
Link: https://github.com/Aditya-0308/Student-Performance-Predictor

##  Features
- Predicts students' final marks using Machine Learning
- Interactive sliders for user input
- Built using Linear Regression
- Displays performance categories
- Deployed on Streamlit Cloud
- Clean and user-friendly interface

## Tech Stack
- **Programming Language:** Python
- **Libraries:** NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
- **Framework:** Streamlit
- **Version Control:** Git & GitHub
- **Deployment:** Streamlit Community Cloud
